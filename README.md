# Linked List
## Singly Linked List Implementation in C

This repository contains a complete, menu-driven console application written in C that demonstrates the fundamental operations of a Singly Linked List.

## Features

The program provides an interactive menu allowing users to perform the following operations:
* **Insertion:**
    * Insert a node at the beginning.
    * Insert a node at a specific position.
    * Insert a node at the end.
* **Deletion:**
    * Delete a node from the beginning.
    * Delete a node from a specific position.
    * Delete a node from the end.
    * Delete the entire list (with a confirmation prompt).
* **Traversal:**
    * Display the current state of the linked list after every operation.
